# Capable — Support

Need help with Capable? Email **capablefitness@proton.me** and include:

1. The version of Capable you're on (Settings → About).
2. Your device and iOS / Android version.
3. A short description of what you tried and what happened.

If you're reporting a bug that involves your data, an exported JSON
backup attached to the email is the fastest way for us to reproduce
it. You can export from Settings → Data → Share backup.

## Common questions

**Does Capable sync across devices?**
No. Capable is local-only by design. Use the JSON export/import to
move data between devices.

**Can I keep my data if I delete the app?**
The OS will remove all of Capable's local data when the app is
uninstalled. Export a backup first if you want to keep it.

**Why does Capable ask for notification permission?**
So your rest timer can fire a single local notification when it
finishes — useful when you put the phone down between sets. Denying
the permission doesn't disable any other feature.
